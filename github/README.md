# Crypto SuperTrend Monitor

这是一个完全基于前端（纯静态页面）的实时监控前 100 加密货币 SuperTrend 指标的应用，数据直接在浏览器中通过 **OKX 官方公开 API** 获取（现货 USDT 交易对，按 24 小时成交额排序）。

## 功能

*   **实时数据**: 每分钟自动抓取市值前 100 的加密货币。
*   **多周期趋势**: 显示 15分钟、1小时、4小时、1天的 SuperTrend (ATR=10, Multiplier=3) 趋势（向上/向下）。
*   **前端展示**: 响应式表格，支持搜索、排序。
*   **后端架构**: Python (FastAPI) + Pandas (数据处理) + APScheduler (定时任务)。
*   **容错处理**: 当 API 不可用时，自动降级使用模拟数据。

## 技术栈

*   **后端**: 无（所有逻辑都在浏览器前端执行）
*   **数据源**: OKX 公共市场数据 API（真实行情与 K 线）
*   **前端**: Vue 3 (CDN), Tailwind CSS (CDN)

## 本地开发与运行

因为是纯前端应用，你有两种使用方式：

1.  **直接双击打开**:
    - 打开 `client/index.html` 即可在浏览器中查看（部分浏览器在 `file://` 模式下可能有 CORS 限制，如果遇到问题，请使用方式 2）。

2.  **使用简单本地静态服务器**（推荐）:
    在项目根目录运行：
    ```bash
    cd client
    python -m http.server 8000
    ```
    然后在浏览器访问 `http://localhost:8000`

## 部署说明 (Linux/Server)

建议使用 Docker 或 Systemd 进行部署。

### 注意事项

*   **排序规则**: 当前依据 OKX 现货 USDT 交易对的 24 小时成交额进行排序，Top 100 为成交最活跃的币种；所有价格与 K 线、趋势均来自真实 OKX 行情数据。
*   **API 限制**: 默认使用 OKX 公共 API。浏览器所在环境需要能访问 `https://www.okx.com`，并且允许该域名的跨域请求。
*   **网络问题**: 如果无法连接 OKX，前端不会生成任何模拟数据，只会在控制台输出错误，此时页面会显示“暂无数据或正在加载…”。*** End Patch ***!
